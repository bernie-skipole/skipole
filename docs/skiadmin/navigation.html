Navigation

The main navigation buttons of the skiadmin web interface:

Along the top:

'Save' which saves your project changes.

'Help' - which shows help messages appropriate to the page displayed.

'Root' - shows the root folder of your project, further buttons link to the pages currently being edited.

When the root folder is displayed, it shows the sub folders and pages within your project which can be edited.

At the left:

'Project' - Opens your project, useful to view as you make changes.

'Admin' - The main admin page.

'Root Folder' - Takes you to the root folder of your project.

'Page Labels' - Enables you to assign text labels to pages.

'TextBlocks' - Blocks of text or static HTML can be predefined, for later insertion into templates.

'Sections' - Blocks of dynamic HTML and widgets can be predefined.

Further buttons will be displayed depending on your position in the system.

Finally an input field where you can choose the label or ident of a page to be edited.
